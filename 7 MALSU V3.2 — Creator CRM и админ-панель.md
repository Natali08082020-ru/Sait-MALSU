# MALSU V3.2 — Creator CRM и полноценная админ-панель MalSu

## Главная цель

Создать полноценную рабочую систему управления проектом MalSu.

Это не просто админка сайта.

Это внутренняя CRM музыкального проекта.

Через неё должно быть удобно:

получать заявки;

вести работу над песнями;

управлять контентом сайта;

хранить информацию по заказам;

отслеживать готовность проектов.

Важно:

НЕ подключать Supabase.

НЕ делать серверную часть.

Использовать localStorage.

Использовать существующий дизайн админки.

# ПОДГОТОВКА К SUPABASE

Вся работа должна идти через сервисный слой.

Запрещено обращаться к localStorage напрямую из интерфейса.

Пример:

ordersService.getAll()

ordersService.save()

albumsService.getAll()

albumsService.save()
